Brush

Brush简介

一个Less库，提供了大量的易用性短标签和方法，用于快速搭建可用网站前端界面和UI。例如本页面是我用一天时间搭建起来的（呵呵，怎么可能）

因为一个网站本身并不需要大量的微调，除非出设计稿的脑子有坑。在同一个网站中，组件与组件之间的距离最好相同或者以倍数相乘。例如在我们目前的项目www.unileverfoodsolutions.com.cn中，外部组件与组件的距离是15px，列表中项与项的距离是7.5px，底部空余距离为30px等。

颜色

Brush内建了多种默认颜色，并通过样式标签来设置字体，背景和边框：

色号 字色 背景 边框

rgba(0, 147, 255, 0.5) .hl-half .hl-half-bg .hl-half-bd
#808080 .gray .gray-bg .gray-bd
#EFEFEF .hoary .hoary-bg .hoary-bd
rgba(239, 239, 239, 0.5) .hoary-half .hoary-half-bg .hoary-half-bd
#FFFFFF .white .white-bg .white-bd
#000000 .black .black-bg .black-bd
rgba( 0, 0, 0, 0 ) .trans .trans-bg .trans-bd
#FE421D .red .red-bg .red-bd
#339933 .green .green-bg .green-bd

.highlight是网站的设计主色调，在使用过程中可以重命名颜色或重新定义颜色。比如使用@font-color : @gray定义文字的基本颜色等。

使用Flex布局

我们十分推荐使用Flex盒模型进行布局，Flex经过几次大的修改，曾命名为box，flex-box，直到现在Chrome以及FireFox还支持以display: box;为基础的盒模型。

盒模型很好的解决了块状组件在容器中的布局问题，尤其是另过去前端开发工作者们备受困扰的上下居中，甚至通过justicy-content: space-between创建了相等间隔的排布，这让我想起了Bootstrap开发者不得不引入row { margin-left: -15px; margin-right: -15px; }才解决此问题（但不得不说当时的解决方法即时在现在看来也很巧妙）。
